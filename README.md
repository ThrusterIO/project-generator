The Thruster ProjectGenerator Tool.

## Install

Via Composer

```bash
$ composer global require thruster/project-generator
```

## Usage

```bash
$ thruster-project
```

## Testing

Run PHP style checker

```bash
$ composer cs-check
```

Run PHP style fixer

```bash
$ composer cs-fix
```

Run all continuous integration tests

```bash
$ composer ci-run
```
